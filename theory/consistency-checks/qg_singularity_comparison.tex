\vspace{4pt}
\footnotesize
\textsuperscript{a}\,Schwarzschild-connected branch ($n\!=\!1$).
  The regular $n\!=\!3$ branch exists but is disconnected
  (L\"u--Perkins--Pope--Stelle, 2015~\cite{LPPS2015}).\\
\textsuperscript{b}\,Regular branch exists mathematically but is
  not selected by gravitational collapse.
  The massive spin-2 ghost renders the theory non-unitary.\\
\textsuperscript{c}\,\textbf{Linearized}: $K \sim 1/r^4$, NEC satisfied,
  Penrose applies, geodesically incomplete.
  \textbf{Nonlinear}: unknown---blocked by Gap~G1 (OP-01).
  Root cause: $\Pi_{\mathrm{TT}} \to \mathrm{const}$ provides zero extra UV smoothing.
  The spectral action nonperturbative argument
  ($S[\mathrm{Schw}] = \infty$, $S[\mathrm{regular}] < \infty$) is suggestive
  but not rigorous.
  \textbf{Unique}: $\alpha_C = 13/120$, $m_0 = \Lambda\sqrt{6}$ are
  parameter-free (SM-derived).\\
\textsuperscript{d}\,LQG: bounce replaces the singular point;
  $m(r)$ is not analytic at the transition surface.
  Immirzi parameter $\gamma_I \approx 0.2375$ fixed by BH entropy.\\
\textsuperscript{e}\,Conditional on RG-improvement scheme.
  Recent work~\cite{WangGongPapa2025} shows resolution requires
  $\alpha > 1$ in $G(\varepsilon) = G_N/[1 + \tilde\omega(G_N^2\varepsilon)^\alpha]$.
  Cutoff identification $k = k(r)$ is an additional input.\\
\textsuperscript{f}\,Fuzzball: no classical interior; the geometry is
  topologically different.\\
\textsuperscript{g}\,Complete only for BPS (extremal, supersymmetric) BHs.
  Non-BPS (Schwarzschild) microstate construction is
  incomplete~\cite{WarnerBena2022}.\\
\textsuperscript{h}\,Action has 0 free parameters ($\alpha_W$ absorbed into $G$).
  The Mannheim--Kazanas solution has 3 integration constants
  $(\beta, \gamma, \kappa)$.\\
\textsuperscript{i}\,Requires $\geq 6$ derivatives.
  The minimal 4-derivative Lee--Wick model has $K \sim 1/r^4$
  (same as Stelle/SCT), not resolved~\cite{Modesto2016LW}.
