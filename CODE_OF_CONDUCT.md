# Code of Conduct

## Our standards

This project is a scientific research workspace. All participants are expected to engage with intellectual honesty, respect for evidence, and professional courtesy.

- Constructive criticism of methods, derivations, and results is encouraged.
- Personal attacks, harassment, and bad-faith argumentation are not tolerated.
- Disagreements should reference specific equations, data, or published literature.

## Scope

This code of conduct applies to all project spaces: Issues, Discussions, Pull Requests, and any other communication channels associated with this repository.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainer. All complaints will be reviewed and will result in a response deemed appropriate to the circumstances.
