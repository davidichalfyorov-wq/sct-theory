# FND-1 Statement

## Core Question
Can the Chamseddine-Connes Spectral Action be formulated directly on a bare discrete causal set so that the resulting construction behaves as a spectral triple in the relevant sense for SCT?

In this archive, `direct synthesis` means a direct map from a causal set to a usable spectral-triple-style operator framework without introducing an intermediate smooth manifold, triangulation stack, or coarse-grained continuum model as the main definition.

## Allowed Ingredients
- A discrete causal set as the starting geometry.
- A candidate operator built from causal-set data.
- A spectral-triple-style algebra/representation pair only if it is derived from the causal set rather than imported as the primary geometry.
- Continuum limits, ensembles, or scaling arguments only as validation tools, not as the definition of the construction.

## Forbidden Escapes
- Treating a smooth manifold as the primary geometry and then pushing the causal set in afterward.
- Declaring success from a toy operator that is not tied back to the causal set data.
- Replacing the standard spectral-action question with an unrelated generalized observable and calling it the same result.
- Using a continuum limit, ensemble average, or truncation trick as the only content of the definition.

## Success Condition
FND-1 is solved only if the archive can exhibit a causally sourced operator framework that supports the spectral-action target without relying on an external geometric scaffold, and that passes the success criteria in `FND1_SUCCESS_CRITERIA.md`.

## Failure Condition
FND-1 fails if the best available route requires an external continuum geometry, an ad hoc background structure, or a candidate operator that cannot satisfy the operator-theoretic and numerical criteria in the companion criteria file.
